CYBERSECURITY THREAT DETECTION SYSTEM

1 • Problem Statement & Solution Overview

Problem Statement:

  • Malicious IP scanning the network.
  • Malware signatures present in network logs.
  • Lateral movement attacks between computers.
  • Large spikes in suspicious traffic.

Solution Overview:

  • Uses KMP pattern matching to detect malware signatures in logs..
  • Uses Hashing (Hash Table) to track failed login attempts per IP.
  • Uses Priority Queue (Heap) to show top suspicious IPs.
  • Uses Graph + BFS to detect lateral movement between computers
  • Uses Sorting to prepare a final report of suspicious activity.

2 • Algorithms & Data Structures

Algorithm / Data Structurepurpose
