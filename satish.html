Case 1 — Digital Advertising & Events Hub

A centralized platform managing digital advertising slots, event bookings, live audience analytics, and stall allocations inside Velora’s Times-Square-style plaza. The system uses ad-slot priority scheduling, automated stall allocation, and drone-based display routing. The hub optimizes advertising performance and event flow with real-time data.

Algorithms Used
  • Hash Table → Stall records & advertiser lookup
  • Max-Heap → Ad-slot priority scheduling
  • Dijkstra → Drone/display routing paths
  • Queue → Live advertisement rotation cycle
