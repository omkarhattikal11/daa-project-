Smart Ambulance Routing System

Optimizes ambulance travel in Velora using real-time road data, distances, and connectivity. Ensures fastest response to emergencies by calculating the shortest path to hospitals.

Algorithms Used
  • Dijkstra → Fastest route calculation
  • Min-Heap → Efficient node selection for Dijkstra
  • Hash Map → Quick hospital data lookup
  • Graph (Adjacency List) → City road representation
